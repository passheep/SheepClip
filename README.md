# SheepClip

SheepClip 是一款面向 Windows 的剪贴板历史和快捷输入工具，适合经常在聊天、客服、开发、办公场景中重复复制、查找、粘贴内容的用户。

## 项目定位

很多用户每天都会反复复制手机号、邮箱、地址、代码片段、截图、文件路径或富文本内容。系统剪贴板只能保留最近一次内容，常用短语也分散在不同文档里，来回查找会打断工作节奏。

SheepClip 试图解决这些问题：

- 自动记录剪贴板历史，减少“刚才复制的内容找不到了”的情况。
- 支持快捷短语，通过 `//` 快速唤起并筛选常用内容。
- 支持文本、图片、文件/文件夹路径、富文本等多类型剪贴板内容。
- 支持双击 Alt 唤起主窗口，尽量减少鼠标操作。
- 支持托盘运行、开机自启、窗口置顶、条目激活后自动粘贴等个性化设置。

## 主要功能

- 剪贴板历史：记录复制过的文本、图片、文件/文件夹、富文本。
- 快捷输入：维护常用手机号、邮箱、地址、回复话术、代码片段等内容。
- 行内触发：在外部输入框快速输入 `//` 后弹出快捷输入窗口。
- 搜索筛选：支持按关键词或标签筛选快捷短语。
- 详情查看：支持查看复制来源、复制时间、图片缩略图、富文本预览等。
- 托盘与窗口：支持隐藏到托盘、失焦关闭、始终置顶、自定义窗口大小位置。
- 新手引导：首次启动提供分步骤功能说明。

## 面向人群

- 客服、运营、销售等需要高频输入固定话术的人。
- 开发者、测试、运维等经常复制命令、代码片段、配置路径的人。
- 经常处理截图、图片、文件路径、富文本内容的办公用户。
- 希望在 Windows 上获得更顺手剪贴板管理体验的用户。
